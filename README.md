# Mapa Cliente

**Formação DICA — Consultor de IA para Pequenos Negócios**

Curso publicado no formato [INEMA.CLUB](https://inema.club) que forma profissionais para diagnosticar, vender, implementar, capacitar e acompanhar soluções de IA aplicada em pequenas empresas.

🌐 **Site publicado:** https://inematds.github.io/mapacliente/

## Método DICA

- **D** — Diagnosticar
- **I** — Implementar
- **C** — Capacitar
- **A** — Acompanhar

## Estrutura

| Trilha | Foco | Módulos |
|--------|------|---------|
| T1 — Mercado & Mentalidade | Por que PME, nova profissão, virada mental, posicionamento | 4 ✅ |
| T2 — Venda Consultiva | Prospecção, abordagem, objeções, fechar diagnóstico | 6 ✅ |
| T3 — Diagnóstico (D) | Entrevista com dono, 5 áreas, mapa de oportunidades | 5 ✅ |
| T4 — Implementação (I) | Playbooks, aprovação humana, 5 pacotes | 5 ✅ |
| T5 — Capacitação (C+A) | Treinar cliente, acompanhamento, recorrência | 4 ✅ |
| T6 — Escala | Padronizar, precificar, portfólio, projeto final | 4 ✅ |

**Total:** 6 trilhas · 28 módulos · 168 tópicos · ~15.000 linhas de HTML.

## Fontes conceituais

Tese educacional + tese operacional + validação de mercado + modelo de treinamento Designer de Soluções de IA + tópicos de conversa + estudos sobre soluções para PME.

## Stack

- HTML estático + Tailwind CDN + Inter (Google Fonts)
- Dark/light mode persistido em localStorage
- Hospedagem: GitHub Pages
